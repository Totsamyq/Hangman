# Виселица (Hangman) — консольная игра на Java

Простая консольная реализация игры «Виселица» на русском языке. Игрок пытается угадать загаданное слово, вводя буквы по одной; при ошибках рисуется классическая виселица, количество попыток ограничено.

## Как это работает

1. При запуске программа спрашивает, хотите ли вы начать новую игру.
2. Слово выбирается случайным образом из словаря (`custom_dic.txt`).
3. Игроку показывается слово, скрытое символами `_` — по одному на каждую букву.
4. Игрок вводит по одной русской букве за раз:
   - если буква есть в слове — она открывается во всех позициях;
   - если буквы нет — засчитывается ошибка и рисуется очередная стадия виселицы.
5. Игра заканчивается победой (слово угадано полностью) или поражением (5 ошибок).
6. После завершения раунда можно начать новую игру или выйти.

## Структура проекта

```
1Progect/
├── src/
│   └── Main.java          # вся логика игры
├── .dictionaries/
│   └── custom.dic.txt     # словарь слов для игры (одно слово на строке)
├── 1Progect.iml           # файл модуля IntelliJ IDEA
└── .idea/                 # служебные файлы IntelliJ IDEA (misc.xml, modules.xml, vcs.xml…)
```

> В репозитории файл словаря называется `custom_dic.txt`, но код (`readDictionary()`) ищет его по пути `.dictionaries/custom.dic.txt`. Перед запуском убедитесь, что файл находится по правильному пути и имеет правильное имя (`custom.dic.txt`), либо поправьте путь в коде.

## Требования

- JDK 17 или новее (используются текстовые блоки `"""..."""`; в `misc.xml` проекта указан `openjdk-26`).
- IntelliJ IDEA (не обязательно, но проект настроен под неё).

## Запуск

### Через IntelliJ IDEA
1. Откройте проект в IntelliJ IDEA.
2. Убедитесь, что папка `.dictionaries` со словарём лежит в корне проекта (рабочая директория запуска).
3. Запустите класс `Main`.

### Через командную строку
```bash
javac -d out src/Main.java
cd out
java Main
```
Убедитесь, что рядом с рабочей директорией (там, откуда запускается `java`) есть папка `.dictionaries/custom.dic.txt`.

## Управление

- `1` — начать новую игру
- `2` — выйти из приложения
- Ввод буквы — любая русская буква (кириллица), одна за раз
